# MyUardecs_mega
MyUardecs_megaの特殊バージョンVer1.2.2

  Uardecs_megaから派生したMyUardecs_megaの特殊バージョンです。
  
  書き換え箇所はMyEthernet2.hとMyEthernetUdp2.hのファイル名のみです。
  
  理由はW5500の内蔵MACアドレス設定機能を有効にするためです。
